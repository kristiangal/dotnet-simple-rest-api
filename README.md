# Basic REST API – C# / .NET

A REST API built in C# as part of learning the .NET ecosystem — covering Web API fundamentals, Entity Framework Core, and JWT authentication.

## About

This project was created to get hands-on experience with C# and .NET after working primarily in JavaScript/TypeScript and Java. It covers the core building blocks of a .NET backend: routing, controllers, EF Core for data access, and JWT-based authentication.

## Features

- RESTful API built with ASP.NET Core Web API
- Entity Framework Core for database access and migrations
- JWT authentication (register / login / protected endpoints)
- Basic CRUD operations

## Tech Stack

| Component | Technology |
|-----------|-----------|
| API | ASP.NET Core Web API |
| ORM | Entity Framework Core |
| Auth | JWT (JSON Web Tokens) |
| Language | C# |

## Notes

Learning project — first hands-on experience with the C# / .NET stack after a background in Node.js and Java.
